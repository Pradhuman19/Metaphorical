# Metaphorical 🛡️

**Metaphorical** is a next-generation Chrome Extension designed to break the "Echo Chamber" of modern journalism. By utilizing the **Gemini 3 Flash** and **Tavily AI Search**, it automatically cross-checks the news article you are reading against the rest of the world's reporting to identify omitted facts, framing biases, and informational gaps.

-----

## 🌟 Key Features

  * **Autonomous Scraping:** Detects when you are on a news site and extracts the core headline and article body, ignoring ads and clutter.
  * **Multi-Perspective Search:** Powered by **Tavily**, the extension fetches 3+ alternative reports from diverse domains (e.g., comparing a Reuters report with Al Jazeera and BBC).
  * **AI-Powered Cross-Check:** Uses **Gemini 3 Flash** to perform a comparative analysis, spotting "Omitted Facts" that the primary source might have missed.
  * **Conflict Level Indicator:** A real-time "Heat Map" for news, labeling stories as Low, Medium, or High conflict based on reporting discrepancies.
  * **Archive & History:** Saves your past analyses to Chrome's local storage, allowing you to build a personal library of verified news.
  * **Modern UI/UX:** A professional, side-panel-first design built with **React** and **Tailwind CSS**, synced via **Stitch**.

-----

## 🧠 The "Sieve" Architecture (How it Works)

Metaphorical operates like a digital sieve for information. Instead of you manually searching for "the other side of the story," the extension performs a three-stage "Agentic" workflow:

1.  **The Eyes (Extraction):** The content script identifies the headline and "cleans" the text.
2.  **The Researcher (Tavily):** An AI-optimized search engine finds the "Three Witnesses"—alternative sources covering the same event.
3.  **The Brain (Gemini):** The LLM reads all four articles simultaneously, filters out the overlapping "noise," and extracts the unique "nuggets" of info present only in the alternative sources.

## 🚀 Getting Started

### Prerequisites

  * **Node.js** (v18 or higher)
  * **Tavily API Key** ([Get it here](https://tavily.com/))
  * **Gemini API Key** ([Get it here](https://aistudio.google.com/))

### Installation

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/pradhuman19/metaphorical.git
    cd metaphorical
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

3.  **Setup Environment Variables:**
    Create a `.env` file in the root directory:

    ```env
    VITE_TAVILY_API_KEY=your_tavily_key_here
    VITE_GEMINI_API_KEY=your_gemini_key_here
    ```

4.  **Build the project:**

    ```bash
    npm run build
    ```

5.  **Load into Chrome:**

      * Open Chrome and navigate to `chrome://extensions/`.
      * Enable **Developer Mode** (top right).
      * Click **Load unpacked** and select the `dist` folder generated by the build.
